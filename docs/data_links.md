# Data Sources: Starship-Part-Forecast-AI

## Recommended Dataset
**Kaggle: NASA Turbofan Jet Engine (CMAPSS)**
- Link: [Predictive Maintenance NASA turbofan Regression](https://www.kaggle.com/beesechurger/predictive-maintenance-nasa-turbofan-regression)
- Description: Multivariate time series data from operational cycles of turbofan jet engines. Focuses on engine degradation over time.

## Instructions
1. Download the dataset from Kaggle.
2. Place the CSV files in the `data/` directory of this repository.
3. Note: The `data/` directory is ignored by Git to avoid large file commits.
